# **App Name**: CannaConnect

## Core Features:

- Feed de fotos/videos: Feed estilo Instagram para que los usuarios publiquen fotos y videos de sus plantas de cannabis.
- Interacción social: Implementar likes, comentarios, seguidores y mensajería privada para la interacción del usuario.
- Perfiles de usuario personalizables: Permitir a los usuarios personalizar sus perfiles con información y mostrar sus cultivos.
- Calendario de cultivo interactivo: Una sección 'Reels' personalizada con un calendario interactivo para organizar las etapas de cultivo (germinación, vegetativo, floración, cosecha).
- Gestión de tareas: Permitir a los usuarios programar y realizar un seguimiento de tareas como riego, poda y fertilización, con recordatorios de notificaciones push.
- Herramienta de análisis de plantas con IA: Herramienta impulsada por IA que analiza fotos de plantas para identificar problemas comunes como deficiencias de nutrientes, plagas y enfermedades, proporcionando soluciones sugeridas.
- Biblioteca de cultivo: Integrar guías, artículos y un diccionario visual sobre técnicas de cultivo de cannabis y resolución de problemas.

## Style Guidelines:

- Color primario: Verde bosque (#386641) para reflejar la vida vegetal y el crecimiento.
- Color de fondo: Oliva oscuro (#4A4E69) para una sensación moderna y sofisticada.
- Color de acento: Verde menta (#6A994E) para resaltar elementos clave y llamadas a la acción.
- Fuente del título: 'Space Grotesk' sans-serif para una sensación moderna y tecnológica. Fuente del cuerpo: 'Inter' para facilitar la lectura.
- Utilice iconos estilizados de hojas de cannabis con un toque tecnológico para representar la marca.
- Diseñe una experiencia de usuario intuitiva similar a Instagram pero adaptada a la comunidad de cultivo de cannabis.
- Implemente animaciones y transiciones sutiles para mejorar la participación del usuario, como cargar contenido nuevo o completar tareas.
